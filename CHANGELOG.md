# Changelog

All notable changes to GLink will be documented in this file.

## 0.1.0 - Unreleased

- Open supported Google Drive placeholder files in embedded Obsidian views.
- Store and manage Google URLs in a local path-based registry.
- Obtain links automatically from Google Drive for desktop on Windows.
- Render supported placeholders as file views and note embeds.
- Migrate saved mappings when linked files or folders are renamed.
- Add an experimental, adjustable dark filter for embedded Google pages.
- Add a compatibility shim for the Google Share split button.
- Add registry import and export tools.
